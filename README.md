# react-native-hexagon

hexagon view

## Installation

```sh
npm install react-native-hexagon
```

## Usage

```js
import { HexagonView } from "react-native-hexagon";

// ...

 <HexagonView
   src="https://picsum.photos/200/300.jpg"
   borderColor="#FFC901"
   borderWidth={6}
   cornerRadius={6}
   width={32}
   height={32}
      />
```
